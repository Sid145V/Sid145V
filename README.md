## 🚀 Projects

### 🔍 AWS Customer Agreement RAG Assistant
> `Python` `FastAPI` `FAISS` `SentenceTransformers` `Google Gemini API` `Streamlit` `SQLite` `SQLAlchemy`

A complete Retrieval-Augmented Generation system that answers legal queries from a multi-page AWS Customer Agreement — built as a real client assignment for VeStaff.

**What I built:**
- PDF ingestion → recursive text chunking → SentenceTransformer embeddings → FAISS vector store → Gemini LLM answer generation
- **Cosine-similarity confidence threshold** to filter irrelevant queries before hitting the LLM — reducing hallucinations and unnecessary API calls
- Modular backend: ingestion service, embeddings service, vector store, RAG pipeline, analytics service — all decoupled
- **SQLite + SQLAlchemy** query logging for full observability
- Streamlit chat interface + Plotly analytics dashboard
- Automated integration test suite covering valid and out-of-context queries
- Live Vercel deployment with Git-integrated CI/CD
